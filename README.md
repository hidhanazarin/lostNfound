### Project Description
LostNFound is a web application that allows users to report, view, and manage lost and found items.
Users can add item details with images, filter items by type, and contact owners easily.
The platform provides a simple and organized interface to help recover lost belongings faster.

### The Problem statement
Lost items are hard to track, and reporting is scattered

### The Solution
LostNFound lets users add, view, and manage lost & found items with images and contact details, all in one place.

### Technologies/Components Used

**For Software:**

## Features

List the key features of your project:
- Add Items: Users can report lost or found items with details like type, title, location, contact, description, category, and optional images.
- View & Filter Items: Browse all items and filter them by type, with the newest items displayed first.
- Delete Items: Remove items easily using their unique ID when they are no longer needed.
- Static Frontend Page: A simple web interface to interact with the platform, including viewing items and their images.
